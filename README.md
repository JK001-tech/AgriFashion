Project Overview

AgriFashion is a Full-Stack Direct-to-Market Platform that connects farmers directly with retailers, eliminating middlemen and improving profits.

This project demonstrates:

Secure user authentication & role-based access control | 
CRUD operations for products and orders | 
API integration between backend and Android frontend | 
MVC architecture & service-layer abstraction | 
Real-time order tracking and notifications.

Tech Stack : 

Backend: Java, Spring Boot, Spring MVC, Spring Security, REST APIs | 
Frontend: Android app (JSON API integration) | 
Database: MySQL, JPA | 
Tools: Git, GitHub, Maven, Postman.
